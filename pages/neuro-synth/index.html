About NeuroSynth

Neuro-sama is an amazing singer, whose singing is mainly enabled through the use of techniques similar to those used in Vocaloid, and later SynthV. However, her voice is not available publicly, so this project aims to create a high-quality Neuro-like voicebank, enabling the creation of covers (and more, like original songs) that don't exist officially.

NeuroSynth does not use SynthV, but is an RVC voice model that can be tuned and controlled in a manner very similar to SynthV.
